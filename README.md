# GeoGuide
GeoGuide AI is an intelligent chatbot designed to help students, beginners, and field researchers understand geology concepts in simple language. It can explain rocks, minerals, geological processes, formations, and even help interpret basic field observations.  Think of it as a “virtual geology tutor + field assistant.”
